## üéØ INTERACTION PATTERNS

- **Color**: Darker shade for primary actions
- **Transition**: 0.2s ease-in-out

## ‚ôø ACCESSIBILITY REQUIREMENTS

### Color Contrast
- **AA Standard**: 4.5:1 for normal text, 3:1 for large text
- **AAA Standard**: 7:1 for normal text, 4.5:1 for large text
- **Interactive Elements**: 3:1 contrast for focus indicators

### Keyboard Navigation
- **Tab Order**: Logical, sequential
- **Focus Indicators**: Visible outline on all interactive elements
- **Skip Links**: Skip to main content, skip to navigation
- **Keyboard Shortcuts**: Documented and accessible

### Screen Readers
- **ARIA Labels**: All interactive elements
- **ARIA Roles**: Semantic HTML + ARIA where needed
- **Alt Text**: All images, icons
- **Form Labels**: Associated with inputs

### Touch Targets
- **Minimum Size**: 44px √ó 44px (mobile)
- **Spacing**: 8px minimum between targets
- **Visual Feedback**: Clear active/pressed states
